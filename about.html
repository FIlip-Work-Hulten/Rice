About

This is a simple calculator built using HTML, CSS, and JavaScript. It allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division. The calculator features a clean and intuitive interface, making it easy for users to input their calculations and view the results.

The design of the calculator is inspired by modern aesthetics, with a focus on usability and visual appeal. The buttons are styled with a glassmorphism effect, giving them a sleek and contemporary look. The background features a gradient that adds to the overall visual experience.

This project was created as a fun and educational exercise in web development. It demonstrates how HTML, CSS, and JavaScript can be combined to create interactive and visually appealing web applications.
